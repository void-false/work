Bootstrap Carousel

Below are the multiple styles of Bootstrap carousel. Bootstrap carousel should have id. Below i am using bs-carousel-1. If a page has more than 1 carousel, then add the new carousel with the id bs-carousel-2. You also need to update the id in, "carousel indicators" section and "carousel control" section. Bootstrap carousel also supports data attributes to customize the slider. They are data-interval for interval between slides, data-pause for pause on hover, data-wrap for looping.
